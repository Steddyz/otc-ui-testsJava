# OTC UI Tests

Автотесты для сайта [OTC.ru](https://otc.ru) с использованием Selenide.

## 📋 Описание проекта

Проект содержит E2E-тесты для проверки функциональности поиска товаров на сайте OTC.ru.  
Реализован поиск товаров по ключевому слову "Принтер" с сохранением результатов в текстовый файл.

## 🛠️ Технологии

- **Java 17**
- **Maven** — управление зависимостями
- **Selenide** — UI-тестирование
- **JUnit 5** — тестовый фреймворк
- **SLF4J** — логирование

## 📁 Структура проекта

```bash
otc-ui-tests/
├── pom.xml
├── README.md
├── src/
│   └── test/
│       ├── java/
│       │   └── com/
│       │       └── example/
│       │           ├── OtcSearchTest.java      # Тесты
│       │           ├── dto/
│       │           │   └── ProductDto.java     # DTO для товара
│       │           ├── pages/
│       │           │   ├── MainPage.java       # POM главной страницы
│       │           │   └── MarketplaceSearchPage.java  # POM страницы поиска
│       │           └── utils/
│       │               └── TestLogger.java     # Утилита логирования
│       └── resources/
│           └── config.properties                # Конфигурация
└── storProducts.txt                            # Результат (создаётся при запуске)
```

## Тесты
- **siteLoadTest()** - Проверка загрузки главной страницы
- **mainPageElementsTest()** - Проверка наличия основных элементов на главной
- **searchProductTest()** - Поиск товаров и сохранение в файл
